# Kinematic Control of a Manipulator via Inverse Kinematics Algorithm

This repository contains the implementation of kinematic control algorithms for a manipulator robot, developed as part of my Bachelor's thesis.

## Overview
The project focuses on inverse kinematics applied to robot motion control, including trajectory tracking and simulation in a virtual environment.

## Contents
* MATLAB Live Scripts with explanations and simulations
* PDF version of the project (recommended for quick overview)
* MATLAB source files

## Key Features
* Inverse kinematics implementation
* Trajectory generation (including square trajectory)
* Simulation and visualization of robot motion in CoppeliaSim

## Tools
* MATLAB / Simulink
* CoppeliaSim (robot simulation environment)
